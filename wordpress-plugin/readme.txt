Installable beta of the Crocoblock Site Factory engine with a Real Estate demo flow.

== Description ==

Crocoblock Site Factory is an infrastructure-style automation engine for WordPress/Crocoblock sites.

This beta includes a Site Factory admin dashboard, Real Estate prompt preview, plan preview, demo generation, run manifests, validation proof, REST run visibility, and optional WP-CLI commands.

The Real Estate demo generates a homepage, properties catalog, property filters, single property pages, a contact page, a menu, and demo content/images.

== Requirements ==

* WordPress 6.0+
* PHP 8.0+
* Kava theme installed and active
* JetEngine plugin installed and active

The beta does not bundle or silently install commercial dependencies.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the WordPress Plugins screen.
3. Open the Site Factory admin menu.
4. Preview and generate the Real Estate demo.
